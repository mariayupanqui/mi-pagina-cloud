¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para estructurar y presentar contenido en la web

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilos fundamental para el desarrollo web que define la presentación y el diseño visual de los documentos HTML o XML

¿Qué es JS?

JS (JavaScript) es un lenguaje de programación fundamental para la web
